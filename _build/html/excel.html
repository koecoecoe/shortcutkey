エクセル編¶

Ctrl + d 上のセルをコピー貼り付け¶

一つ上のセルのコピーと貼り付けをします。

Ctrl + f 検索¶

文字列を検索します。ブラウザや他のアプリでも使用可能です。

Ctrl + h 置換¶

検索した文字列を置換します。Wordやテキストエディターでも使用可能

Ctrl + ;（セミコロン） 現在の日付を挿入¶

アクティブなセルに日付を挿入します。

Ctrl + :（コロン） 現在の時刻を挿入¶

アクティブなセルに時刻を挿入します。

Ctrl + Shift + `(@) 数式だけ表示切替¶

数式の表示非表示を切り替えます。

注釈

数式を削除されているかチェックする時に便利！

Ctrl + PgUp 次のシートに移動¶

次ののシートに移動します。

Ctrl + PgDn 前のシートに移動¶

前のシートに移動します。

Ctrl + 0 列を非表示にする¶

アクティブな列を非表示にします。

Shift + Ctrl + 0 列を再表示にする¶

アクティブな列を再表示します。

Ctrl + 9 行を非表示にする¶

アクティブなアクティブな行を非表示にします。

Shift + Ctrl + 9 行を再表示にする¶

アクティブな行を再表示します。

Ctrl + SPC 列を選択¶

アクティブな列を選択します。

Shift + SPC 行を選択¶

アクティブな行全体を選択します。
